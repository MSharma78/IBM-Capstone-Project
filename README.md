# IBM Data Analyst Capstone Project

## Project Overview

This project was completed as part of the **IBM Data Analyst Professional Certificate**.
The objective was to collect, clean, analyze, and visualize real-world developer survey data to uncover insights about current technology usage, future technology trends, and developer demographics.

The project demonstrates practical data analysis skills including **data collection, data wrangling, exploratory data analysis, data visualization, and dashboard development**.


## Project Objectives

* Analyze the **most widely used programming languages**
* Identify **technologies developers want to work with in the future**
* Explore **developer demographics such as age, education, and geographic distribution**
* Present insights using **interactive dashboards and visualizations**


## Dataset

The analysis uses data from the **Stack Overflow Developer Survey dataset**, which contains responses from thousands of developers worldwide.
The dataset includes information about:

* Programming languages used
* Databases and platforms used
* Web frameworks
* Developer demographics
* Desired technologies for future work


## Project Workflow

### 1. Data Collection

* Retrieved job posting data using APIs
* Imported survey datasets for analysis

### 2. Data Cleaning and Wrangling

* Cleaned multi-select survey columns
* Handled missing values
* Structured data for analysis using **Python and Pandas**

### 3. Exploratory Data Analysis

* Analyzed trends in programming languages, databases, and platforms
* Investigated developer demographics

### 4. Data Visualization

Created visualizations to communicate insights, including:

* Bar charts
* Treemaps
* Bubble charts
* Pie charts
* Stacked bar charts
* Map charts
* Heat maps

### 5. Dashboard Development

Interactive dashboards were created using **IBM Cognos Analytics** to present insights in three sections:

* **Current Technology Usage**
* **Future Technology Trends**
* **Developer Demographics**


## Key Insights

* **JavaScript, SQL, Python, and HTML/CSS** are among the most widely used programming languages.
* **PostgreSQL, MySQL, and SQLite** are the most commonly used database technologies.
* Cloud platforms such as **Amazon Web Services (AWS)** and **Microsoft Azure** dominate platform usage.
* The majority of survey respondents fall within the **25–34 age group** and commonly hold **Bachelor’s or Master’s degrees**.
